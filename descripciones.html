Descripciones

WEB:
En informática, la World Wide Web (La Web) o red informática mundial​ es un sistema que funciona a través de internet, por el cual se pueden transmitir diversos tipos de datos a través del Protocolo de Transferencia de Hipertextos o HTTP, que son los enlaces de la página web.

HTML:
El lenguaje de marcado de hipertexto (HTML) es el lenguaje de marcado estándar para documentos diseñados para su visualización en navegadores web . Define el contenido y la estructura del contenido web. Suele contar con la ayuda de tecnologías como las hojas de estilo en cascada (CSS) y lenguajes de scripting como JavaScript. HTML.

XML:
Similar a HTML, es un lenguaje de programación para estructurar y presentar datos en un formato de texto formateado. Fue desarrollado originalmente para permitir el intercambio de datos independientemente de los sistemas operativos y lenguajes de programación.
